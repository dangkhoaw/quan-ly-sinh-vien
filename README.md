## ✒️ Description

#### Chương trình cho phép quản lý khoản 100 sinh viên/mỗi lớp. Gồm menu sau:

- Thêm sinh viên.

- Sắp xếp danh sách.

- Xóa sinh viên.

- Tìm sinh viên.

- Cấp Mã sinh viên.

- Cấp email.

- In ra danh sách.

- Thoát.

## 📋 Note

- FILE: `function.h` chứa các hàm xử lý chính của chương trình.

- FILE: `main.c` chứa hàm main.

- Để sử dụng chương trình, hãy chạy file `main.c`.
